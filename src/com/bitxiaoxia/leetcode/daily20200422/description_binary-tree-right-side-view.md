### 二叉树的右视图
给定一棵二叉树，想象自己站在它的右侧，按照从顶部到底部的顺序，返回从右侧所能看到的节点值。
```
示例:

输入: [1,2,3,null,5,null,4]
输出: [1, 3, 4]
解释:

   1            <---
 /   \
2     3         <---
 \     \
  5     4       <---
```

### 解题思路
维护一个深度表，深搜即可。
如果有rightNode，加入list。
没有的话加入判断leftNode深度是否大于已有list的长度。如果大于，则加入list。
